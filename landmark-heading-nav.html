Landmarks Headings Keyboard Navigation Bookmarklet for Accessibility Testing

Enable bookmarklet to make Landmarks, HTML5 Section Elements, and Headings keyboard focusable through TAB navigation and use some quick navigation keys:

  • 1, 2, 3, 4, 5, 6 Sets focus to the first Heading Level of each number.
  • b Sets focus to first <header> or role=banner.
  • f Sets focus to first <footer> or role=contentinfo.
  • m Sets focus to first <main> or role=main.
  • n Sets focus to first <nav> or role=navigation.
  • s Sets focus to first <search> or role=search.

Installation

Landmarks Headings Keyboard Navigation Bookmarklet (drag link to bookmarks bar)
